Struktur keseluruhan sistem — apakah ini sudah tepat?

System Architecture
FRONTEND — Next.js 14
/ → Hero + About
/projects → Projects grid
/blog → Blog list
/blog/[slug] → Article
Static Generation (SSG)
Revalidate on CMS update
ADMIN CMS — Payload
/admin → Dashboard
Collections:
• Projects
• Blog Posts
• Skills
• Experience
Frontend fetch data via Payload REST API at build time (SSG) + on-demand revalidation
DATABASE
PostgreSQL
Payload ORM (built-in)
Media uploads → local/S3
INFRA
Docker Compose
frontend + payload + postgres
Deploy: VPS / Railway / Render
FOLDER STRUCTURE
my-portfolio/
├── apps/frontend/ # Next.js 14 + Tailwind
├── apps/cms/ # Payload CMS
├── packages/shared/ # shared TS types
├── docker-compose.yml
└── package.json # workspace root
